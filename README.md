# Crypto ETL Project

ETL-пайплайн для сбора, обработки и анализа криптовалютных данных.

## Стек технологий
- **Apache Airflow**  оркестрация пайплайнов
- **Apache Spark (PySpark)**  обработка больших данных
- **PostgreSQL**  хранение данных
- **Docker**  контейнеризация

## Архитектура
- Данные собираются с CoinGecko API
- Airflow запускает сбор по расписанию
- PySpark обрабатывает и трансформирует данные
- Результат сохраняется в PostgreSQL
- Уведомления в Telegram о статусе
